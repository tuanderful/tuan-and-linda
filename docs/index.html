Requirements

Yes HTML template does not require web server to run, except for some sections which require Ajax call, that is Twitter feed, Instagram and RSVP form. In order to run these 3 sections, you will need to upload Yes theme to server.You need PHP 5.0+ server and PHP function mail() should be enabled. Please note, without having PHP enabled or just running theme from folder, Yes will work too, except the social feeds and RSVP form.

This template is compatible with following browsers:

  • IE9 +
  • Latest Firefox
  • Latest Opera
  • Latest Google Chrome
  • Latest Safari

Setup

In next sections we will explain how to setup Yes Theme. In order to edit html files you will need a code editor (you can use TextMate or CODA for Mac. or If you have a PC with Windows OS you can use Notepad, Dreamviewer or Sublime Text). Like we explained before, you can view this template in any web browser from your computer, but in order for contact form and portfolio ajax to work, you need to run it from server with PHP installed.

HTML Structure

This theme is built on the custom framework, that is small, fast and with grid system that can be easly changed from 1140px to any other. All grid units (columns) are contained within a top level element, with the class name of container. This has a built in "clearfix" so it does not require a clearing element at the end. It also handles centering the grid layout within the page. By default, it has a max width of 1140px. Between container and columns, you can use row element, that has built in clearfix in case you need to clear elements. Here is the general structure.

<div class="container">
   <div class="row">
      <div class="column six">

      </div>
      <div class="column six">

      </div>
   </div>
</div>

There are twelve different sized columns, each of them with its percentage width:

  .column.one    { width: 8.33333%;}
  .column.two    { width: 16.66667%;}
  .column.three  { width: 25%;}
  .column.four   { width: 33.33333%;}
  .column.five   { width: 41.66667%;}
  .column.six    { width: 50%;}
  .column.seven  { width: 58.33333%;}
  .column.eight  { width: 66.66667%;}
  .column.nine   { width: 75%;}
  .column.ten    { width: 83.33333%;}
  .column.eleven { width: 91.66667%;}
  .column.twelve { width: 100%;}

so for example, if you want to split container in half you would use two "column six" columns:

<div class="container">
   <div class="row">
      <div class="column six">

      </div>
      <div class="column six">

      </div>
   </div>
</div>
Each column unit has 15px of padding on the right and left side that, when coupled with the padding from an adjacent column unit (or the grid-container itself), creates a 30px gutter.

Twitter Feed

As it is php file, you need to run your site from server in order for tweets to work, just like contact form. There are few configuration steps you need to do in order for twitter feeds to appear:
 
Step1:
 
In folder "twitter", open file "get_tweets.php" with some text editor, like notepad++ or sublime.
 
Step2:
 
Once you open the file, starting from line 10 you will see some configuration that you need to fill out. Like this:
 
$get_usernames = array(
'twitter_one' => 'twitterapi',
'twitter_two' => 'denismitrovic',
);
 
This is where you need to place twitter usernames for both accounts. And below config is what you need to populate in order for twitter feed to appear.
 
$config = array(
'username' => $twitter_username, // from which twitter account to fetch tweets
'count' => '3', // number of tweets to fetch
'cache' => '3600', // cache calls to twitter api in seconds, use false to disable cache
'api_key' => 'add key here', // Consumer Key (API Key)
'api_secret' => 'add secret here', // Consumer Secret (API Secret)
'access_token' => 'add access token here', // Account Access Token
'access_token_secret' => 'add access token secret here' // Account Access Token Secret
);
 
Thats  api_key, api_secret, access_token, access_token_secret. Username is your username, or username from which user you want tweets to appear. Thats the name that appears after twitter url, for example our username is: "logicathemes", as our link is http://www.twitter.com/logicathemes
 
Step3:
 
For other fields: api_key, api_secret, access_token, access_token_secret, you need to generate application on this link: https://apps.twitter.com/app/new . There is great tutorial on how to generate these fields on this link https://twittercommunity.com/t/how-to-get-my-api-key/7033 or this link https://themepacific.com/how-to-generate-api-key-consumer-token-access-key-for-twitter-oauth/994/
 
Step 4:
 
Once you have these info, fill them in get_tweets.php and save. There is also option on how many tweets you want to display, thats count number and caching time. You should use cache to speed up your site, place it on 10minutes or one hour.
 
Thats it. Once you save "get_tweets.php" with these info, upload it to your site together with whole twitter folder and your new tweets should appear. If you have any problems, do not hesitate to contact us.

Instagram

In order for Instagram to work, you need to edit custom.js file, line 540 - Instagram Section.

You have just two lines that you need to change:

        var instagramTag = 'snowy'; // Add Instagram Tag here
        var instagramClientId = '630b5c9d2cab44e08fb9d014cf00b2b6'; // Add ClientId here

First one is instagram tag that you want images to appear. Second one is Instagram Client ID. 

In order to find Instagram Client ID, follow these steps:

1. Login if your not already.
2. Go to the Instagram Developer page.
3. Click on ”Register your application”.
4. If you don’t already have a application registered, you will now see a form to register a new one. If you instead see a list of ”clients”, click the ”Register new client” at the top right.
5. Set application name to your websites name
6. Type in a description, anything you want
7. Add the url of you site on the two last fields.
8. Hit register.
9. Click ”Manage clients” in the top menu and you will now see you new client in the list and a client id, client secret and more. Copy the client_id and past that in custom.js file.
10. Save custom.js and thats it.

How To

In this section, we will explain how to edit appropriate section or element in corporato theme.

How to change logo in responsive mode

Logo is made out of the text, and for header it only appears in responsive mode. In order to change it, find this code :
<div id="menu-logo" >N <span>&amp;</span> R</div>

and replace it with your names.

Menu

We are using one menu for both desktop and mobile view. In order to change menu, find this section in header:

<nav id="navigation" class="column nine hide-on-mobile">
   <ul>
    <li><a href="#top">Home</a></li>
    <li><a href="#couple">Story</a></li>

    <li><a href="#important-people">Important People</a></li>
    <li><a href="#wedding-gallery">Gallery</a></li>
    <li><a href="#gift-registry">Gift Registry</a></li>
    <li><a href="#guestbook">GuestBook</a></li>
    <li><a href="#contact">Sign Up</a></li>
</ul>
</nav>

Dont forget to hiperlink your menu to appropriate section. So, in order to make the navigation work properly you need to set an anchor to the desired section of the page. Learn more about page anchors here.

Parallax Section

To change image in this section, open up style.css and search for "parallax" class. Just change background-image property with yours and that should be it. Pretty easy, right :)

How does guestbook work?

Once visitor submit guestbook, you should get text to your email. Because this is html theme, which means there is no database in backend to save data, you would need to manualy update guestbooks html in your index.html file, through some text editor.

How to add Background video

Background Video is nice feature to have, but it might slow down your site, so when adding background video, you need to pay attention to that. Also video background does not work for mobile devices and tablets, instead of video, on mobile devices and tables you will see background image. In order to change this image, just change background image for class ".hero"
 
In order to have background video on your site, you should add:
 
<a id="bgndVideo" class="player" data-property="{ videoURL : 'http://youtu.be/R55e-uHQna0' , containment : 'body' , autoPlay : true, mute : false, vol : 3, startAt : 0, opacity : 1}"></a>
After body tag. Change video URL to your video.
 
A detailed documentation on how to use the video background player can be found here
 

How to use the RSVP(Contact) form

Contact form comes out of the box. All that you need to do is open up action.php in some plain text editor, if you do not have one, you can download notepad++ and use it. On line 24 of action.php change "enteryouremail@gmail.com" with your email and thats all.
 

How to change color?

This theme comes with 4 predefined colors, two dark and two light. In order to change color, after style.css you will see default.css style, just change this one with one of the other colors you prefer. You can also create your own color by coping one of the existing ones, renaming it then changing colors to fit the one you prefer., after that just link new color css. 
 

How to add video popup to photo gallery?

Our script only supports youtube and vimeo videos. Example code is this:
 
<div class="block video">
    <div class="image-holder">
        <img class="grayscale" src="images/portfolio/scout4.jpg" alt="image">
    </div>
    <a class="popup-vimeo video-hover mfp-iframe" href="https://vimeo.com/45830194">
        <div class="portfolio-info">
            <h3 class="portfolio-title">This is some title</h3>
            <span class="portfolio-category">Photography</span>
        </div>
    </a>
</div>
 
To use vimeo video, you need to add class class="popup-vimeo video-hover mfp-iframe"  and to use youtube video, just add popup-youtube. More on this you can read here.
 

How to change color

Very simple. After style.css link to color style that you prefer or if you want to create yours, just copy one of the colors and change color to yours. If you are using circles in your theme, you need to open html file and edit its data-scale-color="#191919" to your color. Thats it.
 
 

How to change countdown ?

 

Open custom.js, and search for countdown which starts at line 605. Find: 

 CountDownTimer('12/08/2015 12:00 AM', 'countdown');


and change to your date.
